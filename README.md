# kolz-city-players
 Base smart contract of KOLZ City players
 
 # Compile: 
 npx hardhat compile
 
 # Depoy (needs Wallet Private key, INfura API Key): 
 npx hardhat run scripts/deploy.js --network base

 Get smart contract Address, then:
 # Verify smart contract (needs Basescan API Key): 
 npx hardhat run scripts/verify.js --network base
